# How to use

## from image
```
python ascii_art.py image.png

python ascii_art.py image.png -s [scale 0-1] -t [table 0,1]
```

## from webcam
> _Reccomended to use scale 0 - 0.5_
```
python ascii_webcam.py

python ascii_webcam.py -s [scale 0-1] -t [table 0,1]
```
